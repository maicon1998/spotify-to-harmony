## Requirements
Spotify credentials - [link](https://developer.spotify.com/documentation/web-api/concepts/apps)

- Client id
- Client secret

## Setup
### .env
- Rename .env.example -> .env
- Put your spotify client id and secret in the variables

## Run
- Install the required packages `pip install -r requirements.txt`
- Run `main.py`
